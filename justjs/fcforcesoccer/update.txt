* UPDATE [EN] *
***************

BEFORE ANY UPDATE, WE INVITE YOU TO MAKE A BACKUP 
OF YOUR DATABASE AND ALL THE FILES


VERSION 1.3 to 1.4
-------------------

1) copy all new files to your server
2) open you navigator, go to your website address and follow the update instructions
3) the website is updated


VERSION 1.2 to 1.3
-------------------

1) copy all new files to your server
2) open you navigator, go to your website address and follow the update instructions
3) the website is updated


VERSION 1.1 to 1.2
-------------------

1) copy all new files to your server
2) open you navigator, go to your website address and follow the update instructions
3) the website is updated


VERSION 1.0 to 1.1
-------------------

1) save the configuration file ("include/conf.php") on your local computer
2) remove all existing phpMySport files from your server
3) copy all new files to your server
4) replace the configuration file "conf.php" in the directory "include/"
5) open you navigator, go to your website address and follow the update instructions
6) the website is updated
