== Description ==

A mutational theme for Blocks theme. It's simple and clear, no images and support thread(nested) comments.

Tested on WordPress 2.3.3/2.5.1/2.6.3/2.7.1/2.8, with IE 6/7/8, Firefox 2.0/3.0/3.5, Opera 9.6, Safari 3.2.1 and Chrome 2.0.
Valid XHTML 1.1 and CSS 3!
Best view under 1024×768 or greater.
