    utility $w

    Description

    • $w #

      $w(string) -> Array
      • string (String) – A string with zero or more spaces.

      Splits a string into an array, treating all whitespace as delimiters.

      Equivalent to Ruby's %w{foo bar} or Perl's qw(foo bar).
